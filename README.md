# Methods

- The file ([build_file.py](build_file.py)) prepares the dataset for further tests.

- The module ([rolling_utils.py](utils/rolling_utils.py)) allows computing the cumulative statistical values of the different features.

- The `start_river_analysis()` method ([ML_river.py](ML_river.py)) runs a complete test.
